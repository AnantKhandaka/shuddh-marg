# 🕉️ Shuddhmarg — शुद्ध मार्ग

Jaano kya hai shuddh. Chuno jo hai dharmic.

Shuddhmarg is a public directory that helps users discover halal-certified companies/products and explore dharmic alternatives.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
